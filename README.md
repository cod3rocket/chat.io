## Contribuiting

### Requirements

- [Deno](https://deno.land/) >= 1.29.4
- [Pnpm](https://pnpm.io/) >= 7.x

### Development

#### Clone repository

```bash
git clone git@github.com:cod3rocket/chat.io.git
```

#### Install dependencies

```bash
pnpm i
```

#### Run dev vite server

```bash
pnpm start:dev
```

#### Run dev deno server

```bash
pnpm start:server
```

#### Run end-to-end tests

```bash
pnpm test
```

#### Run unit tests

```bash
pnpm test:unit
```
